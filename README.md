# Instalation
> fist step:

	git clone https://github.com/renanrod4/Periodic-Table-game.git
or 
	
	git clone git@github.com:renanrod4/Periodic-Table-game.git
	

>second step:

	cd Periodic-Table-game

	pip3 install -r requirements.txt

# Open

python3 game.py
